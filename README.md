# GIRISHKOR.COM

Built with React + Vite + TailwindCSS + Lenis + particles.js.

## Stack

- React 18 with hooks and lazy loading
- Vite 5 with code splitting
- TailwindCSS 3 with custom manga theme
- Lenis for smooth scrolling
- particles.js for background effects
- React Router v6
- ESLint + Prettier

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Lint

```bash
npm run lint
npm run format
```
